## Description
Proyecto en NestJs para manejar productos

## Configuraciones del proyecto

## 1- Clonar

```bash
$ git clone https://github.com/estefany-123/teslo-shop.git
```

## 2- Instalar dependencias

```bash
$ npm install
```

## 3- Correr el proyecto 

```bash
$ npm run start:dev

```
